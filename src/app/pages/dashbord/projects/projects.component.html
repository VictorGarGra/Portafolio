Gestor de Tareas Full-Stack (Angular & Spring Boot)

Desarrollé una aplicación web full-stack de gestión de tareas desde cero. Creé una API RESTful robusta con Java y Spring Boot, gestionando la persistencia de datos con Spring Data JPA y migrando la base de datos de SQL Server a PostgreSQL para el entorno de producción. Para el cliente, construí una interfaz de usuario dinámica (SPA) con Angular. Finalmente, automaticé la entrega del proyecto implementando un flujo de despliegue continuo (CI/CD) con Git, Docker, Render y Netlify.

Aplicación Netflix clon (Android & Firebase)

Desarrollé una aplicación de streaming tipo Netflix guiada por un curso. Implementé autenticación de usuarios con Firebase Auth, gestión de datos en tiempo real con Firebase Realtime Database y almacenamiento en Firebase Storage. Integré la Google Drive API para manejo de archivos y desarrollé un sistema de reproducción de videos online/offline con gestor de descargas. Finalmente, diseñé interfaces modernas en XML utilizando Android Studio, Java y AndroidX.

Vehículo Robótico Autónomo con Control Inteligente

Desarrollé un vehículo robótico autónomo capaz de desplazarse de forma inteligente, esquivar obstáculos y operar en distintos modos: control remoto por teclado, interacción mediante comandos de voz y seguimiento de línea. Implementé el control del hardware con Arduino, integrando sensores y un módulo Bluetooth, mientras que la lógica de interacción con el usuario fue programada en Python, permitiendo una comunicación inalámbrica fluida entre el robot y la computadora.
